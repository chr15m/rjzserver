Instructions

  • Start Rjzserver on your computer. Rjzserver creates a small website on your computer that you can reach from your iThing to install scenes.

  • In the File -> Set scene directory you can point Rjzserver to where you keep your scenes. Point it to a directory that contains scene folders with the .rj file name extension.

  • To install scenes, you go to this page with Safari on your iThing. But before you can install scenes, you need to change the proxy settings on your iPhone/iPod to let RjDj install scenes from your own computer instead of rjdj.me.

    Go to "Settings/Wifi/" on the iPhone and pick your network's blue arrow. Scroll down to the proxy settings and change them to:

    • HTTP Proxy on
    • Manual
    • Server: your IP address as reported by rjzserver (like "192.168.1.1")
    • Port: 8314

  • Go to http://rjdj.me on your iThing. You should now see this very page! Tap the install button on a scene, and RjDj will open and download the scene from your computer.
